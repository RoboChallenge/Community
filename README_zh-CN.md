## 如何加入 RC 工作组

按照这 3 个简单步骤加入 RoboChallenge (RC) 工作组，共同为具身智能的未来做出贡献。

### 第一步：阅读治理模式

**RoboChallenge 委员会**的成立旨在确保该倡议在运作上保持透明、公平和长期可持续性，并遵循开放、协作和共识的原则。

> 组织结构由两个关键部分组成：**指导委员会**和若干**工作组**。

![RoboChallenge Organizational Structure](image/readme/structure.svg)

**[点击此处](https://robochallenge.ai/RoboChallenge_Committee_Governance_Model.pdf)阅读更多关于我们的原则和治理模式的信息。**


### 第二步：选择工作组

工作组 (WGs) 是 RoboChallenge 的主要执行单元。每个工作组由负责 RoboChallenge 特定活动方面的贡献者团队组成。每个工作组设有一名秘书，负责组织会议并支持小组的运作。

**当前的工作组包括**：

* **🛠️ 基准与任务设计工作组**：负责设计和准备新的评测集、任务和数据集。
* **🤖 实验室维护工作组**：负责接收机器捐赠、托管机器并进行测试。
* **💻 平台开发工作组**：负责开发和维护 `robochallenge.ai` 网站、API 服务和数据分发。
* **📢 社区参与工作组**：负责通过黑客马拉松、学术合作、线上/线下活动和社交媒体建立知名度。

*注：每个工作组 (WG) 均可提出新倡议，提交提案以供批准，并发布关于其进展和成果的公开报告。*

工作组由来自社区合作伙伴组织的成员组成。

### 第三步：提交申请并加入工作组

有意向的组织和公司可以通过电子邮件发送提案。

请复制下方的模版并发送邮件至 **[partnership@robochallenge.ai](mailto:partnership@robochallenge.ai)**。

```text
我写信申请加入 RoboChallenge 工作组社区。我的信息如下：

组织名称 *：
[您的公司/组织]

电子邮箱 *：
[您的邮箱]

组织介绍 *：
[简要描述您的组织以及您的工作与具身智能的关联。]

想加入的工作组 *：
[选择一个或多个：基准与任务设计 / 实验室维护 / 平台开发 / 社区参与]

计划贡献 *：
[描述您可以带来的资源以及您计划如何与其他合作伙伴协作。]

在 RoboChallenge 的目标：
[解释您希望通过 RoboChallenge 实现什么以及您的长期愿景。]

现有合作伙伴推荐：
[姓名 + 组织 + 来自现有合作伙伴的简短推荐声明。]

组织 Logo *：
[请随邮件附上贵组织的 Logo。]

联系方式 *：
[邮箱、电话、X (Twitter)、微信或其他联系方式。]

更多关于贵组织：
[网站或社交媒体链接。]

此致，
[您的姓名]

```

*> 注：标有 `*` 的字段为必填项。*
